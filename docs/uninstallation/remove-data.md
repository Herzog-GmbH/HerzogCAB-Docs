# Daten entfernen

Die Deinstallation entfernt nur das Programm. Workspace-Daten und globale Benutzerdaten bleiben erhalten - Sie können sie manuell löschen:

* Workspace-Ordner (selbst gewählt)
* `%ProgramData%\Herzog GmbH\Herzog Cab` (globale Benutzer)
* Registry-Schlüssel `HKEY_CURRENT_USER\Software\Herzog GmbH\Herzog Cab` (Fenster-größen u. ae.)
