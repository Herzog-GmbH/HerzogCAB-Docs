# Update-Fehler

**'Repository nicht erreichbar'.** Prüfen Sie die Internet-verbindung und ob die URL `https://herzog-gmbh.github.io/HerzogCAB-Feedback/repository/` im Browser erreichbar ist.

**'Komponente fehlt'.** Schließen Sie das Programm vollständig (auch das Maintenance-Tool) und versuchen Sie es erneut.
