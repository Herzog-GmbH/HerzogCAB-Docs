# Profil wechseln

Sie können zwischen mehreren [Profilen](concept.md) wechseln, um in einem
anderen Arbeitsbereich zu arbeiten.

## So wechseln Sie das Profil

1. **Datei → Einstellungen → Allgemein** öffnen.
2. Auf **Profil wechseln…** klicken und das gewünschte Profil wählen.

Alternativ erfolgt die Profilauswahl beim **Login**.

!!! info "Wechsel über Programmneustart"
    Ein Profilwechsel startet Herzog CAB neu, damit der gewählte Arbeitsbereich
    sauber geladen wird. So wird zugleich verhindert, dass ungespeicherte
    Eingaben verloren gehen – sichern Sie offene Änderungen vorher.
