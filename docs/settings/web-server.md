# Webserver und QR-Code

Der eingebaute Webserver ermöglicht es, Aufträge über QR-Code an der Maschine mit dem Smartphone abzurufen.

Konfigurierbare Parameter:

* Webserver aktiv (ein/aus)
* Port
* IP-Adresse für den QR-Code (manche Rechner haben mehrere)
* Passwort für Zugriffsschutz

!!! warning "Sicherheits-Hinweis"
    Ein offener Webserver im Werks-Netz kann sensible Auftrags-    daten preisgeben. Aktivieren Sie immer den Passwortschutz.
