# Glossary

| Term                     | Meaning                                  |
|--------------------------|------------------------------------------|
| **Bobbin**               | Spool / yarn carrier                     |
| **Carrier / Klueppel**   | Bobbin carrier on a braiding machine     |
| **CmDongle**             | USB licence dongle from Wibu CodeMeter   |
| **Braid angle**          | Angle between strand and longitudinal axis |
| **Horn Gear**            | Gear that guides the carrier paths       |
| **Lay Length**           | Pitch length of the helix                |
| **Linear density**       | Mass per unit length (tex, dtex)         |
| **Maintenance tool**     | Update / uninstall utility               |
| **PCD**                  | Pitch circle diameter                    |
| **Profile**              | Link between user and workspace          |
| **Role**                 | Bundle of permissions                    |
| **Workspace**            | Data folder of a Herzog CAB installation |
