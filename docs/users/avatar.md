# Profilbild ändern

Zu jedem Benutzer kann ein **Profilbild** (Avatar) hinterlegt werden. Es
erscheint unten links in der Navigationsleiste neben Anzeigename und Rolle.

## Speicherort

Profilbilder werden **maschinenweit** abgelegt unter:

```
%ProgramData%\Herzog GmbH\Herzog Cab\auth\avatars\
```

Dadurch sind sie unabhängig vom Workspace verfügbar – der Benutzer hat an jedem
Profil dasselbe Bild.

!!! tip "Wiedererkennung an gemeinsamen Rechnern"
    Gerade an Arbeitsplätzen, die mehrere Bediener teilen, hilft ein Profilbild,
    auf einen Blick zu sehen, wer gerade angemeldet ist.
