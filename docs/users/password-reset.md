# Passwort zurücksetzen

Hat ein Benutzer sein Passwort vergessen, setzt ein **Administrator** es in der
[Benutzerverwaltung](manage.md) neu.

## So setzen Sie ein Passwort zurück

1. **Systemverwaltung → Benutzer** öffnen.
2. Den betroffenen Benutzer in der Liste wählen.
3. Im Editor auf **Passwort zurücksetzen…** klicken.
4. Neues Passwort vergeben und **Speichern**.

![Benutzer-Editor mit der Schaltfläche „Passwort zurücksetzen".](../assets/screenshots/users/benutzer.png)

!!! info "Konto vorübergehend gesperrt?"
    Nach mehreren fehlgeschlagenen Anmeldeversuchen wird ein Konto kurzzeitig
    gesperrt (siehe [Login und Abmelden](login.md)). Die Sperre löst sich nach
    kurzer Zeit von selbst; ein Passwort-Reset ist dafür nicht nötig.
