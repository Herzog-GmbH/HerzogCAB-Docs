# Login und Abmelden

Herzog CAB verlangt vor dem Öffnen des Arbeitsbereichs eine **Anmeldung**. So
ist nachvollziehbar, wer welche Daten bearbeitet, und Rechte greifen pro
Benutzer.

## Anmelden

Beim Start erscheint das Anmeldefenster **Herzog CAB – Anmelden**. Geben Sie
**Benutzername** und **Passwort** ein. Anschließend wird das gewählte
[Profil](../workspace/concept.md) geladen.

!!! info "Erstanmeldung"
    Bei einer frischen Installation legen Sie zunächst ein Administrator-Konto
    an – siehe [Erstes Admin-Konto](../getting-started/admin-account.md).

## Kontosperre

Nach **fünf** fehlgeschlagenen Anmeldeversuchen wird das Konto für kurze Zeit
gesperrt. Das schützt vor dem systematischen Durchprobieren von Passwörtern.
Warten Sie kurz und melden Sie sich erneut an, oder lassen Sie Ihr
[Passwort zurücksetzen](password-reset.md).

## Profil wechseln / abmelden

Ein Wechsel des Profils erfolgt über einen Programmneustart, damit keine
ungespeicherten Eingaben verloren gehen – siehe
[Profil wechseln](../workspace/switch-profile.md).
