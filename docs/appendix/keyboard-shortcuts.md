# Tastenkürzel

| Tastenkombination | Wirkung                                    |
|-------------------|--------------------------------------------|
| ++ctrl+p++        | Drucken                                    |
| ++ctrl+s++        | Speichern                                  |
| ++ctrl+f++        | Suche im Navigationsbaum                   |
| ++alt+left++      | Im Verlauf zurück                         |
| ++alt+right++     | Im Verlauf vorwärts                       |
| ++f1++            | Online-Hilfe                               |
| ++ctrl+q++        | Programm beenden                           |
