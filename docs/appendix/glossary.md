# Glossar

| Begriff                  | Bedeutung                                |
|--------------------------|------------------------------------------|
| **Bobbin**               | Spule (Englisch)                         |
| **Carrier / Klüppel**   | Spulenträger an einer Flechtmaschine    |
| **CmDongle**             | USB-Lizenzdongle von Wibu CodeMeter      |
| **Flechtwinkel**         | Winkel zwischen Strang und Längsachse   |
| **Horn Gear**            | Zahnrad, das die Klüppel-Bahnen führt  |
| **Lay Length**           | Verlegelänge / Schlaglänge             |
| **Lineare Dichte**       | Masse pro Längeneinheit (tex, dtex)     |
| **Maintenance-Tool**     | Update-/Deinstallationswerkzeug          |
| **PCD**                  | Pitch Circle Diameter, Flügelteilkreis  |
| **Profil**               | Verknüpfung Benutzer <-> Workspace      |
| **Rolle**                | Bündel von Berechtigungen               |
| **Workspace**            | Daten-Ordner einer Herzog-CAB-Installation |
