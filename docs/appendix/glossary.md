# Glossar

| Begriff                  | Bedeutung                                |
|--------------------------|------------------------------------------|
| **Bobbin**               | Spule (Englisch)                         |
| **Carrier / Klueppel**   | Spulentraeger an einer Flechtmaschine    |
| **CmDongle**             | USB-Lizenzdongle von Wibu CodeMeter      |
| **Flechtwinkel**         | Winkel zwischen Strang und Laengsachse   |
| **Horn Gear**            | Zahnrad, das die Klueppel-Bahnen fuehrt  |
| **Lay Length**           | Verlegelaenge / Schlaglaenge             |
| **Lineare Dichte**       | Masse pro Laengeneinheit (tex, dtex)     |
| **Maintenance-Tool**     | Update-/Deinstallationswerkzeug          |
| **PCD**                  | Pitch Circle Diameter, Fluegelteilkreis  |
| **Profil**               | Verknuepfung Benutzer <-> Workspace      |
| **Rolle**                | Buendel von Berechtigungen               |
| **Workspace**            | Daten-Ordner einer Herzog-CAB-Installation |
