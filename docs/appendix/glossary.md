# Glossar

| Begriff | Bedeutung |
|---|---|
| **Klöppel** | Spulenträger an einer Flechtmaschine (engl. *Carrier*). |
| **Spule** | Träger des Garns/Drahts auf dem Klöppel (engl. *Bobbin*). |
| **Hohlgeflecht** | Rohr- bzw. schlauchförmiges Geflecht (engl. *tubular braid*). In der App heißt diese Berechnungsgruppe **Hohlgeflecht** (nicht „Rohrgeflecht"). |
| **Geflechtsdichte** | App-Begriff für die Flechtdichte; das zugehörige Maß ist die **Schlaglänge** (engl. *lay length*, auch „Verlegelänge"). |
| **Schlaglänge** | Axialer Vorschub für eine volle Umdrehung eines Strangs um das Produkt. |
| **Feinheit / Titer** | App-Begriff für die **lineare Dichte** – Masse pro Längeneinheit (tex, dtex, den …). |
| **Flechtwinkel** | Winkel zwischen Strang und Längsachse des Produkts. |
| **Fachung** | Anzahl der Fäden je Klöppel. |
| **Horn Gear / Flügelrad** | Zahnrad, das die Klöppel-Bahnen führt. |
| **PCD** | Pitch Circle Diameter (Flügelteilkreis-Durchmesser). |
| **Profil** | Mandant/Arbeitsbereich – verbindet einen Benutzer mit einem Workspace und den Webserver-Optionen. |
| **Workspace / Arbeitsverzeichnis** | Daten-Ordner mit Stammdaten, Aufträgen, Designs und Druckvorlagen. |
| **Rolle** | Bündel von Berechtigungen (Administrator, Bearbeiter, Betrachter …). |
| **CmDongle** | USB-Lizenzdongle von Wibu CodeMeter. |
| **Maintenance-Tool** | Update-/Deinstallationswerkzeug von Herzog CAB. |
