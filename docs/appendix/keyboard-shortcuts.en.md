# Keyboard Shortcuts

| Shortcut          | Action                                      |
|-------------------|---------------------------------------------|
| ++ctrl+p++        | Print                                       |
| ++ctrl+s++        | Save                                        |
| ++ctrl+f++        | Search in the navigation tree               |
| ++alt+left++      | Back in history                             |
| ++alt+right++     | Forward in history                          |
| ++f1++            | Online help                                 |
| ++ctrl+q++        | Quit                                        |
