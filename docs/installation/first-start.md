# Erster Programmstart

Doppelklicken Sie auf das **Herzog CAB**-Symbol auf dem Desktop oder
öffnen Sie das Programm über das Start-Menü (*Herzog > HerzogCAB*).

## Was passiert beim ersten Start?

Beim allerersten Aufruf führt Sie ein **Einrichtungs-Assistent** durch
drei kurze Schritte:

1. **Sprache wählen** – Oberflächensprache festlegen
2. **Workspace einrichten** – wo Ihre Daten gespeichert werden
3. **Erstes Admin-Konto anlegen** – damit das Programm vor unbefugtem
   Zugriff geschützt ist

Danach landen Sie direkt im Hauptfenster und können loslegen.

!!! tip "Sie sind nach dem Wizard – wie geht es weiter?"
    Im Kapitel [Erste Schritte](../getting-started/index.md) finden Sie
    zu jedem der drei Wizard-Schritte eine ausführliche Anleitung mit
    Screenshots, sowie weitere Themen wie **Oberfläche im Überblick**,
    **Navigation und Verlauf** und die eingebaute **Guided Tour**.

## Nächster Schritt

Lesen Sie als Erstes das Kapitel [Workspace einrichten](../getting-started/workspace.md) –
dort wird die wichtigste Einstellung Ihres Systems erklärt.
