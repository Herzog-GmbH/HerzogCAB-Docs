# QR-Code für die Maschine

Pro Auftrag erzeugt Herzog CAB einen QR-Code. Wird er an der Maschine mit dem Smartphone gescannt, öffnet sich automatisch die mobile Auftragssicht über den eingebauten Webserver. Voraussetzung: Webserver ist aktiv - siehe [Webserver und QR-Code](../settings/web-server.md).
