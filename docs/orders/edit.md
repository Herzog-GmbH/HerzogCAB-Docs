# Auftrag bearbeiten

## Auftrag öffnen

In der [Auftragsübersicht](index.md) wählen Sie einen Auftrag in der Liste an
und klicken auf **Öffnen** (oder doppelklicken die Karte). Der Auftrag öffnet
sich im [Auftrags-Editor](create.md) mit allen Tabs.

![Geöffneter Auftrag im Editor (Tab „Kunde").](../assets/screenshots/orders/auftrag-editor-kunde.png)

## Ändern und speichern

1. Wechseln Sie zum gewünschten Tab und passen Sie die Werte an.
2. Sichern Sie mit **Auftrag speichern** (oben rechts).
3. Mit **Zur Auftragsliste** kehren Sie zur Übersicht zurück. Gibt es nicht
   gespeicherte Änderungen, fragt Herzog CAB nach, bevor der Editor verlassen
   wird.

## Status ändern

Den Bearbeitungsfortschritt steuern Sie im Tab **Auftrag** über das Feld
**Status** (Entwurf → Freigegeben → In Produktion → Abgeschlossen). Der Status
erscheint anschließend als Markierung auf der Auftragskarte in der Übersicht.

## Auftrag duplizieren

Markieren Sie einen Auftrag in der Übersicht und klicken Sie auf
**Duplizieren**. Herzog CAB legt eine Kopie an (im Namen mit dem Zusatz
*(Kopie)*). Das spart Zeit, wenn ein neuer Auftrag einem bestehenden stark
ähnelt – Sie passen anschließend nur die abweichenden Werte an.

## Auftrag löschen

Markieren Sie den Auftrag und klicken Sie auf **Löschen**. Vor dem endgültigen
Entfernen erscheint eine Sicherheitsabfrage.

!!! warning "Löschen lässt sich nicht rückgängig machen"
    Ein gelöschter Auftrag kann nicht wiederhergestellt werden. Prüfen Sie vor
    dem Löschen, ob der Auftrag noch benötigt wird – im Zweifel genügt es, den
    Status auf *Abgeschlossen* zu setzen.
