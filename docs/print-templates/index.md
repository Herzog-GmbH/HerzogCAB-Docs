# Druckvorlagen

**Druckvorlagen** bestimmen, wie Aufträge ausgedruckt aussehen – z. B.
Produktionsbegleitschein, Materialliste oder Etikett. Sie gestalten die Vorlagen
im **Druck-Editor** und verwenden sie anschließend beim [Drucken eines
Auftrags](../orders/print.md).

![Der Druck-Editor: Element-Palette links, Seiten-Canvas in der Mitte, Eigenschaften rechts.](../assets/screenshots/print-templates/druck-editor.png)

Vorlagen sind zentral pro Workspace abgelegt (unter `Printouts/templates/`) und
stehen damit allen Bedienern desselben Workspace zur Verfügung.
